# Anchor

Anchor is a requirements delta agent that helps teams detect and understand meaningful changes in product and technical specifications over time.

At a high level, Anchor compares versions of spec content, identifies section-level changes, and classifies impact so teams can focus on what matters most.

The project is built as a TypeScript workspace with:

- `@anchor_app/core`: comparison and classification engine
- `@anchor_app/anchor`: CLI for local and CI workflows

Anchor is currently in active development, with additional features and integrations planned for upcoming releases.
